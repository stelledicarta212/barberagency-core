# GLOSSARY

## Barbería
Tenant dentro del sistema. Cada barbería tiene sus propios datos.

## Barbero
Empleado que presta servicios.

## Servicio
Actividad ofrecida (corte, barba, etc.) con duración y precio.

## Cliente
Persona que agenda una cita.

## Cita
Reserva de un servicio en una fecha y hora específica.

## Slot
Intervalo de tiempo disponible según `slot_min`.

## Horarios
Disponibilidad de la barbería por día.

## RLS (Row Level Security)
Sistema de seguridad que restringe datos por usuario/tenant.

## n8n
Motor de automatización usado para webhooks y lógica externa.

## PostgREST
API automática sobre PostgreSQL.

## JWT
Token de sesión usado para autenticación.

## Soft Delete
Registro no eliminado, marcado con `deleted_at`.
