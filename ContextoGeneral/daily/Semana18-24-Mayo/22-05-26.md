### 3. Lógica de Cierre Automático (JavaScript)
Un script optimizado al final del documento se encarga de mejorar la experiencia del usuario final (*UX*):
- Escucha los clics en cualquier enlace `.nav-link` del menú.
- Verifica mediante clases activas (`.show`) si el menú móvil de Bootstrap está desplegado.
- Invoca la API nativa de Bootstrap (`bootstrap.Collapse`) de forma segura para ocultar/colapsar la barra de navegación inmediatamente después del clic, evitando que el menú abierto tape la pantalla al desplazarse por la landing.

```javascript
// Auto-collapse mobile navbar after link click (V5 / V2)
document.querySelectorAll('#menuNav .nav-link').forEach(link => {
  link.addEventListener('click', () => {
    const menuNav = document.getElementById('menuNav');
    if (menuNav && menuNav.classList.contains('show')) {
      const bsCollapse = bootstrap.Collapse.getInstance(menuNav) || new bootstrap.Collapse(menuNav);
      bsCollapse.hide();
    }
  });
});
```
